# Feature Components

Складні компоненти з бізнес-логікою та composition.

| Компонент           | Опис                             |
| ------------------- | -------------------------------- |
| `EventCard/`        | variant: `default` \| `featured` |
| `GalleryGrid/`      | Masonry grid — desktop (≥1280)   |
| `GallerySlider/`    | Touch carousel — mobile/tablet   |
| `GalleryLightbox/`  | Lightbox при кліку на зображення |
| `SubscriptionForm/` | Email input + submit             |
| `TicketsModal/`     | Modal для кнопки «КВИТКИ»        |

**Залежності:** `ui/*`, `constants/*`, `types/*`, `hooks/*`

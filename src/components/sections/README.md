# Section Components

Секції landing page. Кожна секція — окремий PR / розробник.

| Папка           | Anchor ID  | Макет                               |
| --------------- | ---------- | ----------------------------------- |
| `Hero/`         | —          | Hero + CTA «КВИТКИ» (modal trigger) |
| `Events/`       | `#events`  | «Актуальні події»                   |
| `Gallery/`      | `#gallery` | Grid (desktop) / Slider (mobile)    |
| `Subscription/` | —          | «Підписка» + email form             |
| `About/`        | `#about`   | Placeholder — майбутня секція       |

**Структура файлу секції:**

```
SectionName/
  SectionName.tsx
  SectionName.module.css
  index.ts
```
